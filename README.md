# OpenDiabetic — landing page

*The hive does the work. Your records stay home.*

Static site (no framework, no dependencies) for **opendiabetic.com** on Cloudflare Pages.
Brand-true to the OpenDiabetic asset kit; copy grounded in the board package.

- `index.html` · `style.css` · `assets/` (wordmark, mark, icon SVGs)
- Deploys two ways: **serve root, no build** — or `npm run build` → `dist/` (for a Vite-style
  Pages config). If a deploy looks empty, set Pages → Build command: `npm run build`, Output: `dist`
  (or Build command: *none*, Output: `/`).

*OpenDiabetic is the hive · LocalDiabetic is the healers · DiabeticLedger is the proof.*
